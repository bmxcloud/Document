# 通过 TestFlight 获取客户端

我们建议您在App Store中选择购买正版程式。

### 介绍 <a id="&#x4ECB;&#x7ECD;"></a>

TestFlight是帮助开发者测试客户端的一种方式， 测试者可以使用 TestFlight 来安装预先发行应用到他们 iOS 设备上。

### 目前可以通过TestFlight获取的客户端 <a id="&#x76EE;&#x524D;&#x53EF;&#x4EE5;&#x901A;&#x8FC7;testflight&#x83B7;&#x53D6;&#x7684;&#x5BA2;&#x6237;&#x7AEF;"></a>

* [Kitsunebi](https://testflight.apple.com/join/2w6EF67u)

### 获取方式 <a id="&#x83B7;&#x53D6;&#x65B9;&#x5F0F;"></a>

打开AppStore，搜索Testflight并下载安装

使用Safari打开本页面，点击Kitsunebi客户端的链接，直接选择参与测试计划。

关闭Safari，打开Testflight，即可下载Kitsunebi
